# Exa API Client

Go client for the [Exa AI](https://exa.ai/) Search API using
[resty/v2](https://github.com/go-resty/resty).

## Fetch API Schema

Exa publishes their OpenAPI spec at:

```bash
curl -o exa-openapi-spec.yaml \
  https://raw.githubusercontent.com/exa-labs/openapi-spec/refs/heads/master/exa-openapi-spec.yaml
```
